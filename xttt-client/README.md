### Getting Started

Install all the packages

~~~bash
npm install
~~~

To start the development web server use the following command and turn your browser to [http://localhost:3000](http://localhost:3000) or [http://0.0.0.0:3000](http://0.0.0.0:3000) 

~~~bash
npm start
~~~

To run code lint

~~~bash
npm run lint
~~~

To run code tests

~~~bash
npm run test
~~~

To build the site 

~~~bash
npm run build
~~~
